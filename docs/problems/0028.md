# Problem 0028: Number Spiral Diagonals

## Problem Summary

- Source: Project Euler #28
- Goal: sum the numbers on both diagonals in a 1001x1001 number spiral

## Main Idea

Each concentric layer contributes a simple four-corner sum.

## Primary Solution

Sum the center value and then add the corner totals for every odd side length from `3` to `1001`.

## Alternate Approaches

- Derive the closed-form polynomial directly.
- Build the spiral explicitly and sum the diagonals.

## Implementation Notes

- The layer formula keeps the code tiny and avoids any grid allocation.
- Only odd side lengths matter.

## Complexity

- Time: `O(n)`
- Space: `O(1)`

## Verification

- Known answer: `669171001`
