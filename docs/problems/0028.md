# 题目 0028：数字螺旋对角线

## 问题概述

- 来源：Project Euler #28
- 目标：求出 1001x1001 数字螺旋两条对角线上的数字和

## 核心思路

每一层外圈都能直接写出四个角的和。

## 主方案

先加中心值，再把从 `3` 到 `1001` 的每个奇数边长对应的四个角加起来。

## 其他方法

- 直接推导闭式公式。
- 把整张螺旋图真的构造出来，再求对角线和。

## 实现说明

- 层公式让代码非常短，也不需要分配网格。
- 只有奇数边长会出现在中心螺旋层里。

## 复杂度

- 时间：`O(n)`
- 空间：`O(1)`

## 验证

- 已知答案：`669171001`
