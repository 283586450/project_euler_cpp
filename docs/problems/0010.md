# 题目 0010：质数求和

## 问题概述

- 来源：Project Euler #10
- 目标：求出所有小于 2000000 的质数之和

## 核心思路

最自然的方法就是筛法。

## 主方案

使用埃氏筛找出全部质数，然后把它们累加起来。

## 其他方法

- 逐个做素性测试也能做，但速度更慢。
- 如果边界更大，可以考虑分段筛。

## 实现说明

- `std::vector<char>` 可以让筛法保持简单，也避开 `std::vector<bool>` 的一些怪异行为。
- 外层循环在 `prime <= limit / prime` 时停止，避免 `prime * prime` 溢出。

## 复杂度

- 时间：`O(n log log n)`
- 空间：`O(n)`

## 验证

- 已知答案：`142913828922`
