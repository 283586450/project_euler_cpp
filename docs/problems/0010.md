# Problem 0010: Summation of Primes

## Problem Summary

- Source: Project Euler #10
- Goal: sum all primes below two million

## Main Idea

Use a sieve to mark composite numbers, then accumulate the remaining primes.

## Primary Solution

Build a boolean primality table up to the limit, strike out multiples starting from each confirmed prime, and sum all indices that remain prime.

## Alternate Approaches

- Incremental prime generation works, but a sieve is the clearer choice for this bound.
- Segmented sieve techniques are useful for much larger limits.

## Implementation Notes

- `std::vector<char>` keeps the sieve simple and avoids the quirks of `std::vector<bool>`.
- The outer loop stops at `prime <= limit / prime` to avoid overflow in `prime * prime`.

## Complexity

- Time: `O(n log log n)`
- Space: `O(n)`

## Verification

- Known answer: `142913828922`
