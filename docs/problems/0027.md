# Problem 0027: Quadratic Primes

## Problem Summary

- Source: Project Euler #27
- Goal: find the coefficients `a` and `b` that produce the longest run of primes for `n^2 + an + b`

## Main Idea

Search the coefficient space and count how many consecutive prime values each quadratic produces.

## Primary Solution

Restrict `b` to positive primes, scan the legal `a` range, and measure the run length starting from `n = 0` until the polynomial stops producing primes.

## Alternate Approaches

- Apply stronger number-theory filters before the search.
- Use a prime sieve for repeated primality checks.

## Implementation Notes

- The search is small enough that a straightforward implementation is readable and fast.
- The final result is the coefficient product, not the length itself.

## Complexity

- Time: moderate brute force over a small search space
- Space: `O(1)` aside from the prime table

## Verification

- Known answer: `-59231`
