# 题目 0027：二次素数

## 问题概述

- 来源：Project Euler #27
- 目标：寻找系数 `a` 和 `b`，使 `n^2 + an + b` 产生最长的连续素数序列

## 核心思路

直接搜索系数空间，并统计每组系数能连续产出多少个质数。

## 主方案

限制 `b` 为正质数，扫描合法的 `a` 范围，然后从 `n = 0` 开始统计多项式能连续产生多少个质数。

## 其他方法

- 在搜索前加更强的数论筛选。
- 用质数筛来加速重复素性检查。

## 实现说明

- 这个搜索空间不大，朴素实现已经足够清晰和快速。
- 最终答案是系数乘积，不是序列长度本身。

## 复杂度

- 时间：在较小搜索空间上做中等规模的暴力搜索
- 空间：除素数表外为 `O(1)`
