# Problem 0003: Largest Prime Factor

## Problem Summary

- Source: Project Euler #3
- Goal: find the largest prime factor of `600851475143`

## Main Idea

Repeatedly divide out prime factors instead of checking every integer up to the target.

## Primary Solution

Remove all factors of 2 first, then try odd divisors up to `sqrt(n)`. Any remaining value greater than 1 is itself prime and becomes the final answer.

## Alternate Approaches

- Trial division with a precomputed prime list.
- Pollard Rho or similar factorization methods for much larger inputs.

## Implementation Notes

- The loop condition uses `factor <= value / factor` to avoid overflow.
- The remaining value is promoted to the answer if it survives the loop.

## Complexity

- Time: `O(sqrt(n))` in the worst case
- Space: `O(1)`

## Verification

- Known answer: `6857`
