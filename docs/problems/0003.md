# 题目 0003：最大质因数

## 问题概述

- 来源：Project Euler #3
- 目标：求出 `600851475143` 的最大质因数

## 核心思路

不断除去较小的因子，最后剩下的就是最大质因数。

## 主方案

先试除 `2`，再按奇数因子一路检查到剩余值的平方根；每次成功整除后，都立刻缩小搜索空间。

## 其他方法

- 先预生成质数表，再只对质数做试除。
- 如果输入更大，也可以使用更高级的分解算法。

## 实现说明

- 循环条件使用 `factor <= value / factor`，避免乘法溢出。
- 如果最后还剩一个大于 1 的值，它本身就是答案。

## 复杂度

- 时间：最坏 `O(sqrt(n))`
- 空间：`O(1)`

## 验证

- 已知答案：`6857`
