# Problem 0024: Lexicographic Permutations

## Problem Summary

- Source: Project Euler #24
- Goal: find the millionth lexicographic permutation of the digits `0` through `9`

## Main Idea

Use the factorial number system to select each digit without enumerating all permutations.

## Primary Solution

Treat the permutation index as a mixed-radix number, repeatedly choose the next digit block, and erase the used digit from the candidate list.

## Alternate Approaches

- Generate permutations directly and stop at the millionth result.
- Use a library permutation iterator.

## Implementation Notes

- The factorial table for `0!` through `9!` fits comfortably in 64 bits.
- The lexicographic index is converted to zero-based form before selection starts.

## Complexity

- Time: `O(n^2)`
- Space: `O(n)`

## Verification

- Known answer: `2783915460`
