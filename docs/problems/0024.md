# 题目 0024：字典序排列

## 问题概述

- 来源：Project Euler #24
- 目标：求出 0 到 9 的第 1000000 个字典序排列

## 核心思路

用阶乘进制直接定位每一位。

## 主方案

把排列序号转成阶乘进制，逐位选择剩余数字中的第几个元素。

## 其他方法

- 直接生成所有排列。
- 使用现成的排列迭代器。

## 实现说明

- `0!` 到 `9!` 的阶乘表完全能放进 64 位整数。
- 选择前先把序号转成 0-based 更方便处理。

## 复杂度

- 时间：`O(n^2)`
- 空间：`O(n)`
