# 题目 0005：最小公倍数

## 问题概述

- 来源：Project Euler #5
- 目标：求出能被 1 到 20 的每个整数整除的最小正整数

## 核心思路

答案就是整个区间的最小公倍数。

## 主方案

按顺序把 `1` 到 `20` 逐个并入，持续取 `lcm`，让当前结果始终对已经处理过的数保持可整除。

## 其他方法

- 先分解质因数，再对每个质数取最高幂次。
- 也可以从 20 开始暴力试除，但会慢很多。

## 实现说明

- 欧几里得算法可以让 `gcd` 实现更短、更稳。
- 使用 `std::uint64_t` 就足够承载中间结果。

## 复杂度

- 时间：`O(n log n)`，这里 `n = 20`
- 空间：`O(1)`

## 验证

- 已知答案：`232792560`
