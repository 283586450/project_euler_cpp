# Problem 0005: Smallest Multiple

## Problem Summary

- Source: Project Euler #5
- Goal: find the smallest positive number divisible by every integer from 1 to 20

## Main Idea

The answer is the least common multiple of the range `1..20`.

## Primary Solution

Fold `lcm` across the interval and compute each step with `gcd` to keep the intermediate values minimal.

## Alternate Approaches

- Prime-factorize each integer and take the maximum exponent for every prime.
- Search upward from 20 until a divisible number is found, though that is much slower.

## Implementation Notes

- Euclid's algorithm keeps `gcd` compact and reliable.
- Using `std::uint64_t` leaves enough headroom for the intermediate products.

## Complexity

- Time: `O(n log n)` for `n = 20`
- Space: `O(1)`

## Verification

- Known answer: `232792560`
