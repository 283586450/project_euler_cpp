# 题目 0025：千位斐波那契数

## 问题概述

- 来源：Project Euler #25
- 目标：找到第一个拥有 1000 位数字的斐波那契数

## 核心思路

继续做高精度斐波那契，直到位数达到 1000。

## 主方案

只维护相邻两项的大整数，不断相加并记录当前位数，直到长度达到 1000。

## 其他方法

- 用对数公式先估计项数。
- 使用大整数库或 fast doubling。

## 实现说明

- 斐波那契增长足够慢，字符串加法很好理解。
- 前两项是 `1` 和 `1`。

## 复杂度

- 时间：`O(n * d)`
- 空间：`O(d)`

## 验证

- 已知答案：`4782`
