# Problem 0025: 1000-digit Fibonacci Number

## Problem Summary

- Source: Project Euler #25
- Goal: find the index of the first Fibonacci number with 1000 digits

## Main Idea

Keep Fibonacci values as decimal strings and stop as soon as the length reaches 1000 digits.

## Primary Solution

Add the previous two Fibonacci numbers as big decimal strings and count how many terms are needed before the current value has 1000 digits.

## Alternate Approaches

- Use Binet-style estimates to jump close to the answer.
- Use a big-integer library or fast doubling.

## Implementation Notes

- The sequence grows slowly enough that string-based addition is easy to reason about.
- The first two terms are `1` and `1`.

## Complexity

- Time: `O(n * d)`
- Space: `O(d)`

## Verification

- Known answer: `4782`
