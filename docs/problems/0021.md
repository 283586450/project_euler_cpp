# 题目 0021：亲和数

## 问题概述

- 来源：Project Euler #21
- 目标：求出所有不超过 10000 的亲和数之和

## 核心思路

先求每个数的真因数和，再判断是否满足亲和数条件。

## 主方案

对每个 `n`，设 `m = d(n)`。如果 `m != n` 且 `d(m) == n`，那么 `n` 就是亲和数，应计入总和。

## 其他方法

- 预先把整个范围的因数和都算出来。
- 用质因数分解加速真因数和的计算。

## 实现说明

- 直接枚举因数对 10000 的上限来说已经足够快。
- 亲和数判定会自然排除完全数。

## 复杂度

- 时间：大致为 `O(n sqrt n)`
- 空间：`O(1)`

## 验证

- 已知答案：`31626`
