# Problem 0021: Amicable Numbers

## Problem Summary

- Source: Project Euler #21
- Goal: sum all amicable numbers below 10000

## Main Idea

Compute the sum of proper divisors for each value, then test the amicable pair condition.

## Primary Solution

For each `n`, let `m = d(n)`. If `m != n` and `d(m) == n`, then `n` is amicable and contributes to the total.

## Alternate Approaches

- Precompute divisor sums for the whole range.
- Use prime factorization to compute divisor sums faster.

## Implementation Notes

- The direct divisor scan is fast enough for the 10000 limit.
- The pair check avoids counting perfect numbers.

## Complexity

- Time: roughly `O(n sqrt n)`
- Space: `O(1)`

## Verification

- Known answer: `31626`
