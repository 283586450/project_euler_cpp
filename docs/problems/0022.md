# Problem 0022: Names Scores

## Problem Summary

- Source: Project Euler #22
- Goal: sort the names list and sum each name score

## Main Idea

Parse the quoted name list, sort it, and accumulate `position * alphabetical value`.

## Primary Solution

Load the names data as text, split it into names, sort alphabetically, then compute the score for each name using 1-based position.

## Alternate Approaches

- A streaming parser is possible, but unnecessary here.
- Any approach that preserves alphabetical order and 1-based scoring is equivalent.

## Implementation Notes

- The build downloads the names data into the build tree from a stable mirror so the executable stays deterministic during CI.
- The scoring logic is just an alphabetical sum per name.

## Complexity

- Time: `O(n log n)`
- Space: `O(n)`
