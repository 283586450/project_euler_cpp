# 题目 0011：网格中的最大乘积

## 问题概述

- 来源：Project Euler #11
- 目标：求出 20x20 网格中连续 4 个数字的最大乘积

## 核心思路

把四个方向都扫一遍，直接维护最大值。

## 主方案

对每个起点分别检查横向、纵向、两条对角线，取其中的最大乘积。

## 其他方法

- 先预处理每个方向的窗口乘积。
- 也可以把问题拆成多段局部扫描。

## 实现说明

- 网格直接写在源码里，题目保持自包含。
- `std::uint64_t` 足够承载最大的乘积。

## 复杂度

- 时间：`O(r * c * k)`，这里 `k = 4`
- 空间：`O(1)`

## 验证

- 已知答案：`70600674`
