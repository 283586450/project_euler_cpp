# Problem 0011: Largest Product in a Grid

## Problem Summary

- Source: Project Euler #11
- Goal: find the greatest product of four adjacent numbers in the 20x20 grid

## Main Idea

Check every legal 4-number segment in the grid and keep the maximum product.

## Primary Solution

Scan horizontally, vertically, and along both diagonal directions. Because the grid is small, a direct enumeration is the clearest approach.

## Alternate Approaches

- Precompute rolling products for each direction.
- Use a generalized scanning helper for arbitrary window size.

## Implementation Notes

- The grid is stored explicitly so the problem remains self-contained.
- `std::uint64_t` safely covers the largest product.

## Complexity

- Time: `O(r * c)`
- Space: `O(1)`

## Verification

- Known answer: `70600674`
