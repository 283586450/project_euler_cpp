# Problem 0023: Non-Abundant Sums

## Problem Summary

- Source: Project Euler #23
- Goal: sum the positive integers that cannot be written as the sum of two abundant numbers

## Main Idea

Mark every number that can be formed by adding two abundant numbers, then sum the values that remain unmarked.

## Primary Solution

Generate the abundant numbers up to the limit, mark all valid pair sums, and add the numbers that never get marked.

## Alternate Approaches

- Precompute a boolean abundant table and scan sums with two pointers.
- Use a more optimized sieve for divisor sums.

## Implementation Notes

- The classic 28123 bound keeps the search finite.
- A small `char` table is enough to record representable sums.

## Complexity

- Time: `O(n^2)` over the abundant list in the worst case
- Space: `O(n)`

## Verification

- Known answer: `4179871`
