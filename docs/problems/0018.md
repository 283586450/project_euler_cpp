# Problem 0018: Maximum Path Sum I

## Problem Summary

- Source: Project Euler #18
- Goal: find the maximum sum from top to bottom of the given triangle

## Main Idea

Solve the triangle bottom-up so each row only depends on the row beneath it.

## Primary Solution

Start from the last row, fold the larger child into each parent, and repeat until the top value becomes the answer.

## Alternate Approaches

- Recursive search with memoization.
- A top-down dynamic program that accumulates best partial sums.

## Implementation Notes

- The 15-row triangle is embedded directly in the source to keep the executable self-contained.
- Bottom-up DP keeps the implementation short and easy to audit.

## Complexity

- Time: `O(n^2)`
- Space: `O(n)`

## Verification

- Known answer: `1074`
