# 题目 0001：3 和 5 的倍数

## 问题概述

- 来源：Project Euler #1
- 目标：求出所有小于 1000 且能被 3 或 5 整除的自然数之和

## 核心思路

用等差数列求和公式，避免逐个枚举。

## 主方案

分别求出 3 和 5 的倍数和，再减去 15 的重复部分即可。

## 其他方法

- 直接循环枚举也可以，但不如闭式公式简洁。
- 分别按步长 3 和 5 迭代再去重，逻辑会更绕。

## 实现说明

- 用 `n(n + 1) / 2` 计算前 `n` 个倍数之和。
- 程序直接输出答案，方便本地自检。

## 复杂度

- 时间：`O(1)`
- 空间：`O(1)`
