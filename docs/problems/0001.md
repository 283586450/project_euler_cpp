# Problem 0001: Multiples of 3 or 5

## Problem Summary

- Source: Project Euler #1
- Goal: sum all natural numbers below 1000 that are divisible by 3 or 5

## Main Idea

Use arithmetic-series sums instead of iterating every number.

## Primary Solution

Compute the sum of multiples below the limit for 3 and 5, then subtract the overlap at 15 with inclusion-exclusion.

## Alternate Approaches

- Brute force iteration is fine for a tiny limit, but it does not scale as cleanly.
- A loop over the two step sizes is possible, but the closed form is simpler and faster.

## Implementation Notes

- The helper uses `n(n + 1) / 2` for the first `n` multiples of a factor.
- The program prints the final answer directly, which keeps the executable self-checkable.

## Complexity

- Time: `O(1)`
- Space: `O(1)`

## Verification

- Known answer: `233168`
