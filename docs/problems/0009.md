# Problem 0009: Special Pythagorean Triplet

## Problem Summary

- Source: Project Euler #9
- Goal: find the unique Pythagorean triplet with `a + b + c = 1000`

## Main Idea

Search the feasible `a` and `b` values and derive `c` from the sum constraint.

## Primary Solution

Loop over `a < b < c`, compute `c = 1000 - a - b`, and test whether `a^2 + b^2 = c^2`. The first match gives the answer.

## Alternate Approaches

- Use Euclid's formula to generate primitive triples and scale them.
- Solve the Diophantine system algebraically and search fewer candidates.

## Implementation Notes

- The search space is tiny, so the straightforward nested loop is easy to read and verify.
- The product is computed only after the Pythagorean condition passes.

## Complexity

- Time: `O(n^2)`
- Space: `O(1)`

## Verification

- Known answer: `31875000`
