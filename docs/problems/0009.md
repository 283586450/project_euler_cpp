# 题目 0009：特殊勾股三元组

## 问题概述

- 来源：Project Euler #9
- 目标：找出满足 `a + b + c = 1000` 的唯一勾股三元组，并返回 `a * b * c`

## 核心思路

利用和为 1000 这一条件缩小搜索范围，再直接检查勾股关系。

## 主方案

枚举 `a` 和 `b`，令 `c = 1000 - a - b`，然后直接判断是否满足 `a^2 + b^2 = c^2`。

## 其他方法

- 用欧几里得公式生成本原勾股数，再进行倍增。
- 直接从代数关系出发推导答案。

## 实现说明

- 这个搜索空间非常小，双重循环完全够用。
- 一旦找到第一个匹配三元组，就可以直接返回。

## 复杂度

- 时间：朴素枚举下约为 `O(n^2)`
- 空间：`O(1)`

## 验证

- 已知答案：`31875000`
