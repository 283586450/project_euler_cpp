# Problem 0015: Lattice Paths

## Problem Summary

- Source: Project Euler #15
- Goal: count the routes through a 20x20 grid when only moving right or down

## Main Idea

This is a binomial coefficient problem: choose when to step down among 40 total moves.

## Primary Solution

Compute `C(40, 20)` with multiplicative reduction so the intermediate values stay small.

## Alternate Approaches

- Dynamic programming over the grid.
- Closed-form factorials with arbitrary-precision arithmetic.

## Implementation Notes

- GCD reduction keeps the binomial calculation within `std::uint64_t`.
- The combinatorial view is shorter and clearer than filling a 21x21 DP table.

## Complexity

- Time: `O(k log n)` for `k = 20`
- Space: `O(1)`

## Verification

- Known answer: `137846528820`
