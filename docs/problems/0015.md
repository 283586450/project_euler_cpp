# 题目 0015：格点路径

## 问题概述

- 来源：Project Euler #15
- 目标：统计 20x20 方格中从左上到右下的路径数

## 核心思路

这就是一个组合数问题。

## 主方案

直接计算 `C(40, 20)`，并在中间用最大公约数约分，避免中间值过大。

## 其他方法

- 用动态规划填表。
- 用任意精度整数直接算阶乘。

## 实现说明

- GCD 约分可以让二项式系数保持在 `std::uint64_t` 范围内。
- 组合意义比 21x21 的 DP 表更短、更清楚。

## 复杂度

- 时间：`O(k log n)`，这里 `k = 20`
- 空间：`O(1)`

## 验证

- 已知答案：`137846528820`
