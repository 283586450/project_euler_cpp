# 题目 0007：第 10001 个质数

## 问题概述

- 来源：Project Euler #7
- 目标：找出第 10001 个质数

## 核心思路

逐步生成质数，直到到达目标序号。

## 主方案

维护一个不断增长的质数表，拿每个奇数候选数去试除；当表里累积到 10001 个质数时就停止。

## 其他方法

- 如果已知上界，直接筛素数也可以。
- 更高级的质数生成器对本题来说没有必要。

## 实现说明

- 试除会在 `prime > candidate / prime` 时停止，这样能避免溢出，也能少做一些工作。
- 除了 `2` 以外，只测试奇数候选数。

## 复杂度

- 时间：对本题来说是足够快的亚二次级别
- 空间：`O(n)`，用于保存已经找到的质数

## 验证

- 已知答案：`104743`
