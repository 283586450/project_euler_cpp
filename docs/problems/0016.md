# Problem 0016: Power Digit Sum

## Problem Summary

- Source: Project Euler #16
- Goal: sum the digits of `2^1000`

## Main Idea

Represent the growing power as decimal digits and simulate multiplication by 2.

## Primary Solution

Keep the number in little-endian decimal form, double it 1000 times, then sum the remaining digits.

## Alternate Approaches

- Use a big-integer library.
- Use arbitrary-precision integer support from another language runtime.

## Implementation Notes

- The input is small enough that a simple digit vector is more than sufficient.
- No special-case optimization is needed.

## Complexity

- Time: `O(e * d)`, where `e = 1000` and `d` is the number of digits
- Space: `O(d)`

## Verification

- Known answer: `1366`
