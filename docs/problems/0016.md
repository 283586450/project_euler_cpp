# 题目 0016：幂的数字和

## 问题概述

- 来源：Project Euler #16
- 目标：求 `2^1000` 的各位数字之和

## 核心思路

把不断增长的幂表示成十进制数字，再模拟乘 2。

## 主方案

用小端十进制数组保存当前数值，连续乘 1000 次 2，然后把剩余数字求和。

## 其他方法

- 使用大整数库。
- 依赖其他运行时自带的任意精度整数。

## 实现说明

- 这个规模下，简单的数字向量已经完全够用。
- 不需要任何特殊优化。

## 复杂度

- 时间：`O(e * d)`，其中 `e = 1000`，`d` 是位数
- 空间：`O(d)`

## 验证

- 已知答案：`1366`
