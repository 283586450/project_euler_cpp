# Problem 0006: Sum Square Difference

## Problem Summary

- Source: Project Euler #6
- Goal: find the difference between the square of the sum and the sum of squares for the first 100 natural numbers

## Main Idea

Use closed-form formulas instead of enumerating all terms.

## Primary Solution

Compute `1^2 + 2^2 + ... + n^2` and `(1 + 2 + ... + n)^2` separately, then subtract the former from the latter.

## Alternate Approaches

- Direct loops work for this input size, but the formulas are shorter and clearer.
- Symbolic algebra can derive the same result in one expression.

## Implementation Notes

- The formulas stay in `std::uint64_t`, which is more than enough for `n = 100`.
- The helper functions are `constexpr`, so the compiler can fold the result at build time.

## Complexity

- Time: `O(1)`
- Space: `O(1)`

- Known answer: `25164150`
