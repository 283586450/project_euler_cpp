# 题目 0006：平方和与和平方差

## 问题概述

- 来源：Project Euler #6
- 目标：比较前 100 个自然数的“和的平方”和“平方和”

## 核心思路

两项都有闭式公式，因此可以直接算差值。

## 主方案

分别使用 `1 + 2 + ... + n` 的等差数列公式，以及 `1^2 + 2^2 + ... + n^2` 的平方和公式，再做相减。

## 其他方法

- 直接暴力枚举对 `n = 100` 也完全可行。
- 用递推或循环也可以，但公式更短、更清晰。

## 实现说明

- 公式本身就能把问题压缩成常数时间。
- 结果很小，轻松放进 64 位整数。

## 复杂度

- 时间：`O(1)`
- 空间：`O(1)`

## 验证

- 已知答案：`25164150`
