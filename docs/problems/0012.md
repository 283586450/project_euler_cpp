# Problem 0012: Highly Divisible Triangular Number

## Problem Summary

- Source: Project Euler #12
- Goal: find the first triangular number with more than 500 divisors

## Main Idea

Generate triangular numbers incrementally and count their divisors until the threshold is exceeded.

## Primary Solution

Use the factorization structure of `n(n+1)/2` to count divisors efficiently, then stop at the first triangle number with divisor count above 500.

## Alternate Approaches

- Factor every triangle number directly.
- Precompute primes and use them for factorization.

## Implementation Notes

- Splitting `n(n+1)/2` into coprime parts keeps the divisor count calculation compact.
- The search ends quickly enough that no special cache is needed.

## Complexity

- Time: roughly `O(k sqrt n)` for the discovered range
- Space: `O(1)`

## Verification

- Known answer: `76576500`
