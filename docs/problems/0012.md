# 题目 0012：高度可分三角数

## 问题概述

- 来源：Project Euler #12
- 目标：找到第一个拥有超过 500 个因数的三角数

## 核心思路

三角数可以逐个生成，而因数个数可以通过质因数分解来计算。

## 主方案

依次生成三角数，并用因数计数函数判断是否已经超过阈值。

## 其他方法

- 直接对每个三角数试除分解。
- 先筛出足够多的质数，再用它们做分解。

## 实现说明

- `n(n+1)/2` 可以拆成两个互质部分，因数统计会更紧凑。
- 搜索范围并不夸张，因此不需要额外缓存来凑复杂度。

## 复杂度

- 时间：大致为 `O(k sqrt n)`，取决于发现答案前的搜索范围
- 空间：`O(1)`

## 验证

- 已知答案：`76576500`
