# 题目 0004：最大回文积

## 问题概述

- 来源：Project Euler #4
- 目标：找出两个三位数相乘得到的最大回文数

## 核心思路

从大到小枚举因子，并尽早剪枝。

## 主方案

按降序搜索三位数因子，持续记录当前最优回文积；当理论上限已经不可能超过当前最优值时，就提前结束。

## 其他方法

- 枚举所有乘积，再检查是否回文。
- 先生成回文数，再反向分解成两个三位数因子。

## 实现说明

- 只枚举 `b <= a` 可以避免重复乘积。
- 用一个简单的数字反转 helper 就足够判断是否回文。

## 复杂度

- 时间：最坏 `O(n^2)`，但实际会因为剪枝少很多
- 空间：`O(1)`
