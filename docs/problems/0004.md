# Problem 0004: Largest Palindrome Product

## Problem Summary

- Source: Project Euler #4
- Goal: find the largest palindrome made from the product of two 3-digit numbers

## Main Idea

Search the products in descending order and stop early whenever the remaining candidates cannot beat the current best value.

## Primary Solution

Iterate `a` from `999` downward and `b` from `a` downward, test each product for palindromicity, and prune the search once the maximum possible product in a loop can no longer improve the answer.

## Alternate Approaches

- Enumerate all products and sort them.
- Generate palindromes directly and factor them into two 3-digit numbers.

## Implementation Notes

- Restricting `b <= a` avoids duplicate products.
- A small digit-reversal helper is enough to test palindromes.

## Complexity

- Time: `O(n^2)` in the worst case, with pruning in practice
- Space: `O(1)`

## Verification

- Known answer: `906609`
