# Problem 0013: Large Sum

## Problem Summary

- Source: Project Euler #13
- Goal: compute the first ten digits of the sum of 100 50-digit numbers

## Main Idea

Add the numbers digit by digit from right to left, keeping only the carry-aware total that matters for the leading digits.

## Primary Solution

Store the input as text, accumulate each column into a fixed-size digit buffer, propagate carries, then extract the first ten digits from the final sum.

## Alternate Approaches

- Use a big-integer library.
- Sum the values with a language runtime that already supports arbitrary precision integers.

## Implementation Notes

- A text-based representation keeps the problem self-contained and avoids external dependencies.
- The fixed-size buffer is larger than the final sum, so the carry propagation is straightforward.

## Complexity

- Time: `O(n * m)` for `n` numbers of `m` digits
- Space: `O(m)`

## Verification

- Known answer: `5537376230`
