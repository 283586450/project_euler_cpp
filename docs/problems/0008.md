# Problem 0008: Largest Product in a Series

## Problem Summary

- Source: Project Euler #8
- Goal: find the largest product of 13 adjacent digits in the given 1000-digit number

## Main Idea

Scan every length-13 window and compute its digit product.

## Primary Solution

Store the 1000-digit input as a string, multiply the digits in each window, and keep the maximum product.

## Alternate Approaches

- A sliding window with zero skipping can reduce repeated work.
- Dynamic updates are possible, but unnecessary at this scale.

## Implementation Notes

- `std::uint64_t` is enough for the largest 13-digit product here.
- The implementation stays simple because the input size is small.

## Complexity

- Time: `O(n * k)`, where `n` is the digit count and `k = 13`
- Space: `O(1)`

## Verification

- Known answer: `23514624000`
