# Problem 0029: Distinct Powers

## Problem Summary

- Source: Project Euler #29
- Goal: count distinct values of `a^b` for `2 <= a <= 100` and `2 <= b <= 100`

## Main Idea

Normalize each power by its prime-exponent signature and store the signature in a set.

## Primary Solution

Factor each base once, multiply the exponent vector by the chosen power, and deduplicate the resulting signatures.

## Alternate Approaches

- Use big integers and compare exact values.
- Compare logarithms, though that is not exact enough for deduplication.

## Implementation Notes

- Prime-factor signatures avoid overflow and preserve exact equality.
- The search space is small enough that a set-based solution is fine.

## Complexity

- Time: `O(n^2 * f)`
- Space: `O(n^2)` in the worst case for the signature set

## Verification

- Known answer: `9183`
