# 题目 0029：不同的幂

## 问题概述

- 来源：Project Euler #29
- 目标：统计 `2 <= a <= 100` 且 `2 <= b <= 100` 时不同 `a^b` 的个数

## 核心思路

把每个幂写成唯一签名，再放入集合去重。

## 主方案

对每个底数分解质因数，把指数乘上幂次，形成标准化签名后放入集合中去重。

## 其他方法

- 直接算出大整数再比较。
- 用对数近似比较大小，但无法保证去重精确。

## 实现说明

- 质因数签名可以避免溢出，也能保持精确比较。
- 这个搜索空间足够小，集合去重很合适。

## 复杂度

- 时间：`O(n^2 * f)`
- 空间：最坏情况下为 `O(n^2)` 的签名集合
