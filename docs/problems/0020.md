# Problem 0020: Factorial Digit Sum

## Problem Summary

- Source: Project Euler #20
- Goal: sum the digits of `100!`

## Main Idea

Compute the factorial as a decimal digit array and sum the digits at the end.

## Primary Solution

Multiply the growing big integer by each factor from `2` to `100`, then add up the final decimal digits.

## Alternate Approaches

- Use a big-integer library.
- Use prime factorization plus digit-sum reasoning, though that is less direct here.

## Implementation Notes

- The arithmetic is done entirely with decimal digits, so overflow is not a concern.
- The multiplication loop is simple enough for this input size.

## Complexity

- Time: `O(n * d)`
- Space: `O(d)`

## Verification

- Known answer: `648`
