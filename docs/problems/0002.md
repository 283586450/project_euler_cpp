# 题目 0002：偶数斐波那契数

## 问题概述

- 来源：Project Euler #2
- 目标：求所有不超过 4000000 的偶数斐波那契数之和

## 核心思路

偶数斐波那契数本身也满足递推关系，因此可以直接跳过所有奇数项。

## 主方案

直接维护偶数子序列，使用 `E(n) = 4E(n-1) + E(n-2)` 递推并累加，直到超过上限。

## 其他方法

- 先生成完整斐波那契序列，再过滤偶数项。
- 如果上限更大，也可以用矩阵快速幂或 fast doubling。

## 实现说明

- 前两个偶数斐波那契数是 `2` 和 `8`。
- 整个过程都在 `std::uint64_t` 中完成，足够应对本题范围。

## 复杂度

- 时间：`O(k)`，其中 `k` 是上限内偶数斐波那契数的个数
- 空间：`O(1)`

## 验证

- 已知答案：`4613732`
