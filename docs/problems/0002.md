# Problem 0002: Even Fibonacci Numbers

## Problem Summary

- Source: Project Euler #2
- Goal: sum the even Fibonacci terms not exceeding four million

## Main Idea

The even Fibonacci numbers form their own recurrence, so we can skip all odd terms.

## Primary Solution

Track the even-valued subsequence directly with `E(n) = 4E(n-1) + E(n-2)` and accumulate values until the limit is exceeded.

## Alternate Approaches

- Generate the full Fibonacci sequence and filter even terms.
- Use matrix exponentiation or fast doubling if the limit were much larger.

## Implementation Notes

- The first two even Fibonacci values are `2` and `8`.
- The loop stays entirely in `std::uint64_t`, which is more than enough for this bound.

## Complexity

- Time: `O(k)`, where `k` is the number of even Fibonacci terms up to the limit
- Space: `O(1)`

## Verification

- Known answer: `4613732`
