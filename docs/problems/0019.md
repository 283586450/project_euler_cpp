# Problem 0019: Counting Sundays

## Problem Summary

- Source: Project Euler #19
- Goal: count how many Sundays fell on the first day of a month during 1901-2000

## Main Idea

Advance month by month and track the weekday of each month start.

## Primary Solution

Start from January 1, 1900, step through every month up to December 2000, and count the months whose first day lands on Sunday.

## Alternate Approaches

- Use a date library.
- Convert each month start to an absolute day count and reduce modulo 7.

## Implementation Notes

- Leap years follow the normal Gregorian rules.
- The calendar simulation stays small enough to avoid any extra abstraction.

## Complexity

- Time: `O(years * months)`
- Space: `O(1)`

## Verification

- Known answer: `171`
