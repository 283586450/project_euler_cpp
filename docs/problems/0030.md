# 题目 0030：五次数字幂

## 问题概述

- 来源：Project Euler #30
- 目标：求出所有能够写成其各位数字的五次幂之和的数的总和

## 核心思路

候选空间有上界，因此直接枚举并检查每个数即可。

## 主方案

先给出一个足够大的上界，再逐个判断数字的五次幂和是否等于自身。

## 其他方法

- 用数字动态规划。
- 先从 `n * 9^5` 推导更精确的边界。

## 实现说明

- 预先计算 `0^5` 到 `9^5` 可以减少重复运算。
- 题目中的 `1` 不计入最终总和。

## 复杂度

- 时间：`O(n * d)`
- 空间：`O(1)`

## 验证

- 已知答案：`443839`
