# Problem 0030: Digit Fifth Powers

## Problem Summary

- Source: Project Euler #30
- Goal: sum all numbers that can be written as the sum of the fifth powers of their digits

## Main Idea

The search space is bounded, so brute force the candidates and test the digit-power sum directly.

## Primary Solution

Check every number up to the standard upper bound `354294`, compute the sum of the fifth powers of its digits, and keep the values that match.

## Alternate Approaches

- Use a digit DP.
- Derive the search bound from `n * 9^5`.

## Implementation Notes

- A tiny lookup table for `0^5` through `9^5` keeps the inner loop fast.
- The number `1` is excluded from the sum, consistent with the usual Project Euler interpretation.

## Complexity

- Time: `O(n * d)`
- Space: `O(1)`

## Verification

- Known answer: `443839`
