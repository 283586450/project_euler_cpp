# Problem 0017: Number Letter Counts

## Problem Summary

- Source: Project Euler #17
- Goal: count the letters used when writing `1` through `1000` in words

## Main Idea

Count each number by applying the British English spelling rules directly.

## Primary Solution

Use small lookup tables for ones, teens, and tens, then apply the `hundred` and `and` rules for values above 100.

## Alternate Approaches

- Generate the words as strings and strip spaces and hyphens.
- Hardcode each range, though that is less maintainable.

## Implementation Notes

- `1000` is handled separately as `one thousand`.
- The implementation counts letters only, not spaces or hyphens.

## Complexity

- Time: `O(1)`
- Space: `O(1)`
