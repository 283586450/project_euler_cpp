# 题目 0017：数字字母计数

## 问题概述

- 来源：Project Euler #17
- 目标：统计 1 到 1000 的英文数字写法总共用了多少个字母

## 核心思路

直接按英式英文的拼写规则统计每个数字的字母数。

## 主方案

使用 ones、teens 和 tens 的小表，再对 100 以上的数字套用 `hundred` 和 `and` 规则。

## 其他方法

- 先生成完整英文字符串，再去掉空格和连字符。
- 也可以硬编码每个区间，但维护性更差。

## 实现说明

- `1000` 单独处理为 `one thousand`。
- 这里只统计字母，不统计空格和连字符。

## 复杂度

- 时间：`O(1)`
- 空间：`O(1)`
