# Code of Conduct

## Our Pledge

We expect everyone participating in this project to help maintain a respectful,
inclusive, and constructive environment.

## Expected Behavior

- Be respectful in discussion and code review.
- Focus feedback on ideas, implementations, and evidence.
- Give credit when you build on someone else's work.
- Be patient with learners and contributors from different backgrounds.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Dismissive or abusive language
- Deliberate trolling or intimidation
- Sharing private information without permission

## Enforcement

Project maintainers may remove or edit content, request changes, or restrict participation when behavior violates this code of conduct.

If you need to report a concern, open a private contact channel through the repository maintainer or platform moderation tools.
